Дані про користувачів зберігаються в базі даних H2.
Ідентифікація поточного користувача виконується на основі даних, збережених в сессії.

Для зручності забезпечено доступ до консолі бази даних H2 (для перегляду схеми БД  і вмісту)

Містить: 
- форму реєстрації, 
- форму авторизації користувачів, 
- особисту сторінку користувача,
- сторінку зі списком користувачів.
